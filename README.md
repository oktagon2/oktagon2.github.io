# README

## Dokumentation lokal anschauen
* Dazu braucht es den Docsify Server. Dieser kann über "Terminal > Run Task... > Run docsify server" gestartet werden
* Dieser Server startet in einer eigenen Eingabeaufforderung. Wenn man den Server nicht mehr barucht, muss man ihn von Hand beenden. Dazu die Eingabeaufforderung anklicken und den Task wie üblich mit Ctrl+C beenden.
* Läuft der Docsify Server, so kann man die Dokumentation mit "Run and Debug > Launch Chrome against localhost" lokal anschauen.
* Ändert man was an der Dokumentation, so wird die Webseite automatisch aktualisiert.
